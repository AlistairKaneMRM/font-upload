# example-rollout

A scaffold-time example so the `validate-incoming` workflow has something to run on the first PR. The `.otf` files here are empty placeholders — this is not a real font and should never be merged through the real rollout pipeline.

Once Phase 5 (the actual rollout workflow) is in place, delete this directory and replace it with `examples/example-rollout/` as a reference for future requesters.
